# RLEC (Resource Limit Exceeded Counter)

A shared add/subtract tally website.

## What users need to do

Open the site URL in a browser.

No app install and no server startup is required for visitors.

## How shared tally works

The page uses a public cloud counter API (`api.counterapi.dev`) so all devices see and update the same count.

## Publish once, then share URL

You can host this as a static site (for example GitHub Pages, Netlify, or Vercel).
After that, everyone only needs to open the URL.
